HORAIRES NC ⇄ FRANCE — Installation sur iPad
================================================

Ce dossier contient 2 fichiers à garder ENSEMBLE :
  - nc-france-horaires.html
  - icon-180.png

ÉTAPE 1 — Transférer les fichiers sur l'iPad
----------------------------------------------
Choisis une méthode :
  • AirDrop depuis un Mac : sélectionne les 2 fichiers → clic droit → Partager → AirDrop
  • iCloud Drive : dépose les 2 fichiers dans un dossier iCloud Drive, il apparaîtra
    automatiquement dans l'app "Fichiers" de l'iPad
  • Mail : envoie-toi les 2 fichiers en pièce jointe, puis sur l'iPad, appuie sur
    chaque pièce jointe → "Enregistrer dans Fichiers"

Dans tous les cas, place les 2 fichiers dans le MÊME dossier sur l'iPad
(par exemple "Fichiers" → "Sur mon iPad" → crée un dossier "Horaires").

ÉTAPE 2 — Ouvrir le fichier dans Safari
------------------------------------------
1. Ouvre l'app "Fichiers" sur l'iPad
2. Appuie sur "nc-france-horaires.html"
3. Il s'ouvre en aperçu — appuie sur l'icône de partage (carré avec flèche vers le haut)
4. Choisis "Ouvrir dans Safari" (si l'option n'apparaît pas directement, fais un
   appui long sur le fichier puis "Partager" → "Safari")

ÉTAPE 3 — Ajouter à l'écran d'accueil
-----------------------------------------
1. Une fois la page ouverte dans Safari, appuie sur l'icône de partage
   (carré avec flèche vers le haut, en haut ou en bas selon le modèle d'iPad)
2. Fais défiler et choisis "Sur l'écran d'accueil"
3. Donne un nom court si tu veux (ex : "Horaires FR-NC")
4. Appuie sur "Ajouter"

Une icône (horloge turquoise) apparaît sur l'écran d'accueil. En appuyant dessus,
l'appli s'ouvre en plein écran, sans barre Safari — comme une vraie application.

À NOTER
-------
- Ne déplace pas et ne supprime pas "nc-france-horaires.html" après avoir créé le
  raccourci : l'icône continue de pointer vers ce fichier.
- Aucune connexion internet n'est nécessaire une fois les fichiers sur l'iPad
  (seules les polices d'écriture se chargent en ligne au premier lancement).
